# 曲谱图片批量重命名工具 - 需求文档

## 项目概述

自动识别曲谱图片中的调号和曲名，批量重命名文件。支持两种使用模式：
- 分步处理：先识别输出 CSV，手动校对后再重命名
- 一步完成：识别+重命名一次完成

## 功能需求

### 1. 使用模式

#### 模式一：分步处理（推荐）
1. **识别阶段** (`recognizer.py`)
   - 对图片进行 OCR 识别
   - 输出 CSV 文件（不重命名）
   - CSV 不含"新文件名"列

2. **校对阶段**
   - 手动编辑 CSV，修正调号和曲名

3. **重命名阶段** (`renamer.py`)
   - 读取 CSV 根据调号和曲名重命名文件
   - 支持参数：`python renamer.py <文件夹编号>` 或 `python renamer.py <相对路径>`

#### 模式二：一步完成
- `recognize_and_rename.py` - 识别+重命名一次完成

### 2. 图片文件处理
- 支持的图片格式：`.jpg`, `.jpeg`, `.png`
- 过滤已重命名的文件（以调号开头的如 `C-曲名.jpg`）
- 保留仅曲名命名的文件（如 `曲名.jpg`），支持后续修复调号

### 3. 曲名识别
- 使用 EasyOCR 识别图片上部 40% 区域
- 按从上到下顺序取第一个非调号信息文字块作为曲名
- 排除内容：包含"调"或数字的文字、制谱/编曲/作曲/记谱/演奏等信息
- **曲名输出时自动去除空格**

### 4. 调号识别（多种方案依次尝试）
- **方法1**：EasyOCR 文本正则匹配
  - 匹配格式：`1=C`, `1=G`, `1=#C`, `1=bD` (ASCII)
  - 匹配格式：`1=♯C`, `1=♭D` (Unicode 符号)
  - 匹配格式：`C调`, `bB调`, `#F调` (X调格式)

- **方法2**：Tesseract 识别左上角 30% 区域
  - 匹配单独的大写字母调号 (C/D/E/F/G/A/B)

- **方法2.5**：Tesseract 检测调号前方升降号
  - 在调号字母前方区域检测 `#`, `♯`, `b`, `B`, `6`, `0`, `♭`, `ь` 等符号

- **方法3**：Tesseract 识别曲名下方区域
  - 匹配 `C调`, `D调` 等格式

- **方法4**：Tesseract 识别整个上部 40% 区域（备选）

### 5. 文件重命名
- 格式：`调号-曲名.jpg`（如 `C-童年.jpg`）
- 仅曲名：`曲名.jpg`（如 `时间煮雨.jpg`）
- 重名处理：添加原文件名后缀

### 6. 输出
- 重命名后的图片文件
- `识别结果.csv`：
  - recognizer.py: 原文件名、调号、调号识别方法、升降号检测文本、曲名、状态
  - recognize_and_rename.py: 上述 + 新文件名

## 非功能需求

- Python 3.8+
- 需要 Tesseract OCR 安装配置
- 处理结果输出 CSV 报告便于问题排查

## 依赖

```
easyocr
pytesseract
opencv-python
pillow
tqdm

```

## 运行命令

```bash
# 分步处理
python recognizer.py <文件夹编号>           # 生成 CSV
python renamer.py <文件夹编号>                # 重命名（支持：0, test3, test3/识别结果.csv）

# 一步完成
python recognize_and_rename.py <文件夹编号>
```
